\section{Menu}
\subsection{Overview}
The Menu classes inherit the other major abstract class of the program: \code{Menu\_Abstract.java}. This provides an interface for all the menus (and potentially for the GUI). The abstract class defines how the menu should be displayed, the input to access other entries, builds the menu entries which the user can interact with, and call methods (such as other menus, get/set methods, calculations), etc.
